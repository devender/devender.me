* The coolest thing I saw today was **[sentilla](http://blog.sentilla.com/2008/05/sentilla-announces-worlds-smal.php)** , I have been looking for something like this for so long and add [Phidgets](http://www.phidgets.com/) to it and the possibilities are limitless.
* Sun wants to [**JavaFX**](https://devender.wordpress.com/wp-content/uploads/2008/05/javafx.pdf) on every device that runs Java from Mobile to BlueRay some of the demos crashed but it does look cool check it out [here](https://openjfx.dev.java.net/), I was not too impressed with the Key note I felt last year’s was much better.
* [**Fortress**](https://devender.wordpress.com/wp-content/uploads/2008/05/fortress.pdf) a new language from Sun, Motto is Fortress should do to Fortran what Java did to C. Uses some functional programming paradigms written to run on top of the JVM still in very early stages, I seriously doubt if this will take off syntax was weird who well it will inter operate with other Java Libraries is still to be seen.
* [More Effective Java](https://devender.wordpress.com/wp-content/uploads/2008/05/effectivejava.pdf) was awesome !! I am definitely going to get the new [edition](http://www.amazon.com/Effective-Java-2nd-Joshua-Bloch/dp/0321356683/ref=pd_bbs_sr_1?ie=UTF8&s=books&qid=1210136901&sr=8-1), revised and a lot of stuff rewritten.
* You can get a [free edition](http://developer.mapquest.com/) from MapQuest to develop your own MashUps and do cool stuff.
* [Cacheonix](www.cacheonix.com) is a new Cache Product Find a bug and get a free License!!
* To get GeoSpatial information on Wifi devices [Sky hook](http://www.skyhookwireless.com/) was recommended to me.
* Use [OpenGeo.org](http://opengeo.org/) or [DeCarta](http://www.decarta.com/) to do maps and mashups instead of using from google and friends since there are no restrictions and will always be free and will never add Ads.
* Intel is collaborating with Java to make the JVM highly optimized on Intel architecture out of the box. [Link](http://softwarecommunity.intel.com/articles/eng/1267.htm)
* [Fork/Join](http://www.infoq.com/news/2008/03/fork_join) and [JSR 166](http://gee.cs.oswego.edu/dl/concurrency-interest/) for all the new concurrency features in 1.7
